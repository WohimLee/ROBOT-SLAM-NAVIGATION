# 系统框架

这个系统支持3中传感器数据的输入：
- 视觉: 必备传感器, 双目、RGB-D 二选一
- 激光: 选配传感器
  - 单线: 数据格式 LaserScan
  - 多线: 数据格式 PointCloud
- 里程计: 由一个单独的 ROS 节点（Odometry Node）来提供
  - 轮式里程计
  - 视觉里程计
  - 激光里程计

最后需要将视觉传感器、激光传感器、机器人底盘等之间的安装位置通过 tf 关系输入系统
```
/odom -> /base_footprint
/odom -> /base_link
/base_link -> /camera_link
/base_link -> /laser_link
```

由于各个数据是通过不同话题异步
